Dichotomy

You have a function f with his expression. You want to find the zeros of the function. Considering the fact that you know briefly where a zeros of f is localised. You take two real a and b as f(a)>0 and f(b)<0. To approach the zero of f between a and b, we will iterate number_of_iteration times this procedure:
-We calcul the middle between a and b, m=(a+b)/2.
-if f(m)>0, then we change the value of a to m.
-Else f(m)<0, then we change the value of b to m.
Besides, we return the value of f(m) and m. Finally, we can check if the algorithm returns a suitable value of the zeros of f between a and b.

Considering the function f:x->exp(x)+x*x*x, we want to calcul the zeros of it. Let's say we calcul the zeros between a = -10 and b = 10, after 20 iterations of the procedure explained above, we pbtain the zeros value -0.772886. Moreover, the value of f(-0.772886) = 10^(-15). So, the result of the algorithm is quite satisfying. However, this method considers some hypothesis on the function f. If the function f doesn't have any zeros between a and b, there is more than one zeros between a and b or any other tricky situations, the algorithm will not return converge or return something weird.
