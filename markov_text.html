This text in french is a weird text because the succesion of words seems to be logical but all sentences don't have any sens. In fact, we can't remark it by reading the text attentively. Firstly, we will deal the question : how does it work ? Then, we will enumerate some methods to improve the program to consolidate the sens of sentences.

How does it work ?

We give to our program a text . The first step of our program is to learn how to mix words logically. For that, we use a Markov chain containing each words of the text called previous_word with the occurences of each words after the previous_word. For example, the text " Je code en c++." will have this markov chain :
{("Je",{("code",1)}),("code", {("en",1)}),("en" , {("c++", 1)}), ("c++", {(".",1)}),} The first element ("Je",{("code",1)}) mean after the word "Je", there is 1 occurence of the word "code". Let's say that another sentence came after the previous sentence : "Je code en python.".
The markov chain will be now :
{("Je",{("code",2)}),("code", {("en",2)}),("en" , {("c++", 1),("python",1)}), ("c++", {(".",1)}), ("python", {(".",1)})}
The program is learning the succesion of words with the text. It's learning phase. So, we do it for all the text that has been given by the user. In my code, i name that function learn() opening the file source, filling the markov chain rightfully and returning the number of words in that file. The function adding elements in the markov chain is named add_item. She takes two elements, the word concerned and the word after it. There is a lot of disjonction of cases to deal with. See the code if you are interested(link at the end).
Then, when the step of learning is finished, the program has to generate a text from this markov chain. The function called write() take a word and choose randomnly a word from the possible word after it.
